*Systematically build your personal brand with positioning, content strategy, and cross-platform growth.*

A strategic personal branding assistant that helps professionals, creators, and entrepreneurs build a compelling online presence. From defining your unique positioning to executing a multi-platform content strategy, grow your influence and attract opportunities.

## Features

| Feature | Description |
|---------|-------------|
| Brand Positioning Engine | Defines your unique value proposition, target audience, and differentiation strategy |
| Content Strategy Planner | Creates platform-specific content calendars aligned with your brand pillars |
| Bio & Profile Optimizer | Crafts compelling bios, headlines, and profiles for LinkedIn, Twitter, and other platforms |
| Thought Leadership Builder | Generates thought leadership content including articles, threads, and speaking topics |
| Audience Growth Tactics | Provides proven strategies for follower growth, engagement, and community building |
| Brand Audit Tool | Analyzes your current online presence and identifies improvement opportunities |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-personal-brand-builder-skill
```

## Usage

```
define brand [background]  # Create your brand positioning strategy
content plan [platform]    # Generate platform-specific content calendar
optimize profile [platform] # Optimize your social media profiles
audit brand [name/URL]     # Analyze current brand presence
grow audience [platform]   # Get growth tactics for specific platform
```
